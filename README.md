# my_projec_yolov8n
随着科技的进一步发展，电动车走进了家家户户，成为了人们主要的交通工具，但骑行者是否正确佩戴头盔而发生交通安全隐患，已经引起交通安全部门的重视。交通安全部门希望广大骑行者在骑行过程中佩戴头盔可以有效的保护驾驶者的生命安全。为此解决人工监管电动车驾乘人员头盔佩戴检测效率低下、费时费力等问题，本文提出一种改进YOLOv8n的电动车驾乘人员头盔佩戴检测方法，以YOLOv8为基准框架，在网络模块上，在颈部网络部分引入更有效的CBAM注意力机制和SE注意力机制提高模型的检测性能，增强了对不同骑行者是否佩戴头盔特征的提取能力融合局部和全局特征信息，增强对不同环境的佩戴头盔的检测能力。
